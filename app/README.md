# nft-frame-app
Electron application for NFT Frame.

## Get Started
```
$ yarn install
$ yarn run start
```

## Packaging
Build package to install app for each platform.

### For Mac & Windows
```
$ yarn run package
```

### For Mac
```
$ yarn run package-mac
```

### For Windows
```
$ yarn run package-win
```

### For Windows 32bit
```
$ yarn run package-win32
```
